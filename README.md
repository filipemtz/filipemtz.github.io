
# Filipe Mutz Website

This website was developed with [Quarto <img width=15px src=assets/img/misc/quarto.png />](https://quarto.org/).

## Updating the Website

Edit the necessary files. To visualize the site locally, use the following command:

```bash
quarto preview
```

After finishing the changes, commit and push to the git repository. After that, publishing to github pages using the following command:

```bash
quarto publish gh-pages
```
