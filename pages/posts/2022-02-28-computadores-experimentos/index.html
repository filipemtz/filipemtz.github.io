    <h2> Procedimento de Acesso </h2>
    1) Enviar nome completo, orientador, nome de usuário do acadêmico e e-mail para filipe.mutz@ifes.edu.br (se for
    docente, enviar o SIAPE). <br>
    Como resposta serão realizadas as seguintes ações:
    <ul>
        <li>O usuário receberá permissão de acesso à rede do laboratório.</li>
        <li>Será criado um usuário nos computadores.</li>
        <li>O usuário será adicionado na agenda do Google para reserva dos computadores.</li>
    </ul>
    A senha da conta nos computadores será enviada por e-mail.<br><br>
    2) Após receber o e-mail confirmando que o cadastro para acesso à rede foi realizado, realizar a configuração da
    VPN descrito mais abaixo.<br>
    3) Adicionar um evento na agenda do Google para reserva do computador. No evento informar o nome, orientador, o
    computador que será usado e o período de uso.<br>
    4) Para acessar as máquinas, use o comando ssh como ilustrado abaixo: <br>
    <div class="code">
        ssh joao@172.19.113.7
    </div>
</div>

<div class="col-md-12">
    <h2> Configuração do Ambiente de Desenvolvimento </h2>
    0) Projetos diferentes podem demandar diferentes versões de bibliotecas. Para evitar que um usuário instale
    versões que quebrem as versões dos outros, os usuários devem utilizar <a
        href="https://docs.python.org/3/tutorial/venv.html">ambientes virtuais python</a> (ou contêiners docker se
    forem usadas outras linguagens). Para criar um ambiente virtual python, use o comando abaixo a partir da home do
    usuário: <br>
    <div class="code">
        python -m venv venv
    </div>
    1) Para ativar o ambiente virtual, use o comando abaixo: <br>
    <div class="code">
        source ~/venv/bin/activate
    </div>
    Para evitar ter que digitar esta linha sempre, edite o arquivo ~/.bashrc e adicione o comando ao final.<br>
    2) Pacotes podem ser instalados usando o comando pip. Para instalar os pacotes mais comuns, primeiro atualize o
    pip:<br>
    <div class='code'>
        python -m pip install pip --upgrade
    </div>
    3) E em seguida instale as bibliotecas: <br>
    <div class='code'>
        python -m pip install numpy opencv-python wheel testresources tensorflow seaborn matplotlib scikit-learn
        torch torchvision torchaudio xgboost tqdm scipy requests Pillow pydicom autopep8
    </div>
